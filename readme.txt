== Description ==
You don't need to set target blank to open link in new tab in each file just install this plugin and enjoy. If you want to set [target ="blank"] by class or id you can also set this option. Feel free to contact me at farhanhaider73@gmail.com



= Plugin advantage =

* No need to add target="_blank" in each file and no need to give option to open link in new tab.
* Easy to configuration.
* Unblockable.
* Plugin supports Chrome, Firefox, Safari and IE

== Installation ==


	1. Go to wp-admin -> Plugins > Add New -> and search for 'Aw target blank'.
	2. Click to install.
	3. Once installed, activate.
	
OR

Manual Installation:

	1. Upload the entire `Aw-target-blank` folder to the `/wp-content/plugins/` directory.
	2. Activate the plugin through the 'Plugins' menu in WordPress.
	3. Go to `wp-admin -> AWTB
